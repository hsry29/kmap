# 카카오 좌표 일괄 변환 입력 파일
#
# 형식:
#   ## 테마이름        ← 같은 카테고리는 한 그룹으로 묶임
#   - 장소 검색어       ← 카카오에 그대로 키워드 검색됨
#   - 장소 검색어 지역힌트  ← 동/구/지하철역 등을 함께 적으면 매칭 정확도가 올라감
#
# 사용:
#   npm run resolve:names
#
# 결과:
#   src/data/places.resolved.js 에 저장됨. 검토 후 places.js 로 옮기세요.
#
# 매칭 안 되는 장소가 있다면 카카오맵에서 정확한 명칭을 확인 후 키워드를 고쳐 주세요.
